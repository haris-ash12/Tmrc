Microsoft SharePoint Components
SharePoint Server

Can be found in Enterprise and Standard edition for both internet and intranet sites offering industrial-grade communication, collaboration, and content management.

Business Connectivity Services

Users can directly access the data from the SharePoint to get the details as Business Connectivity Services is connected to Line of Business systems.

Blogs

Employees of the firm write a blog to make it accessible for the whole firm. Further, all others can participate in a discussion of the blog post entry to make communication effective.

SharePoint Foundation

Part of the Windows Operating System permits communication, collaboration, and content management.

Sites

SharePoint Sites contains Document Libraries, Lists, Wikis, Blogs, and Discussion Boards. All are compatible with content-management.

Discussion Boards

Provides a portal for online discussion to a firm. So the members of the firm can post questions and receive the replies.

Key Performance Indicator

Presents instant visualization of business information to measure the growth of the firm and can be entrenched in Share Point Websites.

Lists

Lists available in SharePoint manage the data in rows and columns to access the data at a later time.

Report Builder

A useful end-user tool designed to develop SQL Server Reporting Services (SSRS) reports according to the user’s requirements.

InfoPath from Services

InfoPath forms can be integrated into SharePoint Web Site.

Document Libraries

Users are able to store content in SharePoint through Document Libraries.

PerformancePoint

Offers the ability to build advanced scorecards, dashboards, and reports.

Excel Services

Excel document can be embedded into SharePoint Web Site.

Wikis

Wikis give chance to community members to update the content consistently.

Dashboard Designer

Effective end user tool to create PerformancePoint Content.

SQL Server Reporting Services

Works with SharePoint environment to manage the SSRS reports in SharePoint.

Scorecard

Offers graphs, charts, numbers and visualization tools to keep the track of targeted goals.

Dashboard

Presents the status of key information on the basis of individual tasks.
